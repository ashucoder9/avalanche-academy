Parse Table of contents from markdown/mdx content.

> [You can use the remark plugin directly](/docs/headless/mdx/toc)

## Usage

Note: If you're using a CMS, you should use the API provided by the CMS instead.

```ts
import { getTableOfContents } from 'fumadocs-core/server';

const toc = await getTableOfContents(page.body.raw);
```

### Output

An array of [TOCItemType](/docs/headless/mdx/toc#output) is returned.
