## Usage

No arguments needed.

<InstallTabs>
<Tab value="npm">

```bash
npm create fumadocs-app
```

</Tab>
<Tab value="pnpm">

```bash
pnpm create fumadocs-app
```

</Tab>
<Tab value="yarn">

```bash
yarn create fumadocs-app
```

</Tab>
</InstallTabs>

### Options

You can choose a built-in content source, including Contentlayer and Fumadocs MDX.

It can also configure Tailwind CSS for styling.
